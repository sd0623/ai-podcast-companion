# Sample Audio

For the POC demo, use a **5–10 minute conversational podcast clip** (interview or essay-style works best).

## Quick options

1. **In-app sample button** — The upload page includes "Try a sample clip" which fetches a short MP3 for testing playback and UI.

2. **Your own file** — Export a short segment from any podcast app, or record yourself talking for 2–5 minutes to test transcription and chat.

3. **Public domain** — Search for Creative Commons or public domain spoken-word audio on [Internet Archive](https://archive.org/details/audio).

## Tips for a good demo

- Clear speech beats noisy music intros
- English content works best with the default Whisper `base` model
- 5–10 minutes is enough to test proactive comments (first one after ~90s of playback)
